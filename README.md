# Thai Ground Coffee Guide

An independent buyer's guide comparing Thailand's major ground coffee brands — from iconic commercial staples to award-winning specialty roasters.

## Stack

- **Static site** — pure HTML/CSS/JS generated from Jinja2 templates + `coffee-data.json`
- **Python build pipeline** — data-driven page generation, image processing
- **icrawler** — image acquisition from Bing + Baidu
- **Nginx** — static file serving with gzip + appropriate cache headers
- **Docker** — containerized, runs on port `8420`

## Setup & Build

### 1. Install Python dependencies

```bash
pip install icrawler Pillow Jinja2
```

### 2. Download images (optional — skip to build if images already exist)

```bash
python scripts/download_images.py
```

This uses Bing + Baidu via icrawler with rate limiting and UA rotation. Expect ~20–40 minutes for all brands.

### 3. Process images

```bash
python scripts/process_images.py
```

Converts all raw downloads to WebP (max 1 MB), generates 300×300 thumbnails, updates `coffee-data.json` with filenames.

### 4. Build the static site

```bash
python scripts/build_site.py
```

Generates all HTML pages in `dist/` and copies CSS, JS, and processed images.

### 5. Run locally (Docker)

```bash
docker compose up --build
```

Open http://localhost:8420

### 6. Deploy to vps2

```bash
# Push to GitHub first
git init
git add .
git commit -m "Initial build: Thai ground coffee comparison guide"
gh repo create thai-coffee-guide --public --source=. --push

# Deploy
docker context use vps2
docker compose up -d --build
```

## Design Principles

- **Editorial/magazine aesthetic** — Playfair Display headlines, Source Serif 4 body text
- **No purple anywhere** — Palette: deep brown `#2C1810`, cream `#FAF7F2`, terracotta `#C4652A`
- **No AI design tropes** — no gradient blobs, no pill buttons everywhere, no centered-everything layouts
- **Mobile-first responsive** — Thai users are heavily mobile
- **Performance** — lazy-loaded WebP images, gzip compression, appropriate cache headers
